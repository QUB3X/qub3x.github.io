Intro

Sfire.js is a simple to use mobile menu based on jQuery and CSS Animations.
It's still in development, so there might be some bug.
Scroll to check out an example, then implent it!

Install

It's really simple and fast the installation of this plugin.
  1. First, add these few lines of code to the <head>:
    <link rel="stylesheet" href="sfire.css">
    <link rel="stylesheet" href="animate.css">
    <script src="sfire.js"></script>
    Be sure the link is correct! This should work if files are in the same directory of your html file.
  2. Then, add this to your body, in the top:
    <div class="sfire-navbar">
     <div class="sfire-btn sfire-btn-fixed sfire-btn-left"></div>
     <ul class="sfire-menu">
       <li><a href="#">Home</a></li>
       <li><a href="#">About</a></li>
       <li><a href="#">Contact</a></li>
       <li><a href="#">Other stuff</a></li>
       <li><a href="#">Stuff stuff stuff</a></li>
     </ul>
    </div>

    Place in .sfire-navbar what you want to show only on mobile phones or tablets (or every screen with witdh < 991px.
    Replace menu links ("home, about, ecc.").
  3. Add a <script>$('.sfire-btn').sfireBtn()</script> in the bottom of the body. Eventually, you could place all the script tags in the bottom of the body, because page might load faster (I learned this looking at Twitter Bootstrap example page).
  4. Replace <div class=".sfire-btn"></div> with whatever trigger you want (just remember to remove that div from the html page), or keep the default menu leaving it.
    If you use the default menu icon, in the same element with the class .sfire-btn add these classes to change the style of the button:
    .sfire-btn-left
    Align the button in the top-left corner.
    .sfire-btn-right
    Align the button in the top-right corner.
    .sfire-btn-fixed
    Fix the button in the selected corner to show it when page is scrolled.
    .sfire-btn-static
    The button won't move when the page is scrolled.
    5. And that's it. You've done!
