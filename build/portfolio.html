Portfolio

Here is a quick list of projects I worked on in the last years:

  • Heroes API

    Heroes API is an unofficial REST API I made for Blizzard’s Heroes of the Storm, a game I often played with my friends. Since three years after launch the game still doesn’t provides its own API, I decided to put together a scrapper to gather as much data as possible from various sites, which ironically don’t provide any APIs, too. I did all I could to maximize the efficiency of the API by caching the retrieved data, in order to not impact too much on scrapped websites resources. It’s hosted on Glitch.com.

  • HotS Discord Bot

    HotS Discord Bot is a very quick Discord bot I coded to test and use the above Heroes API. Even though is now offline, it provided players with basic stats in a quick and useful format.

  • DnD Spell Cards

    Dnd Spell Cards was my first project with ReactJS, to help me and my friends to keep track of their spells during our Dungeons & Dragons sessions.

  • CAHOnline

    CAHOnline was my and @YoDevilHots take on a web port of the famous Cards Against Humanity. We used it as an occasion to test AngularJS and Socket.io in a small project. The final result was playable, with some occasional bug due to players disconnection or CSS device-related glitches.

  • LIM

    LIM was one of the first projects I and @YoDevilHots eventually finished during high-school. It was a desktop app built on ElectronJS and vanilla HTML5 Canvas that aimed to provide an extremely simple interface to help not-really-tech-savvy teachers to draw on interactive whiteboards.

  • Other Projects

    I have of course used other programming languages, including Python and recently Swift, although I’ve never actually finished a project to a point I considered myself satisfied with it. iOS Development was quite interesting and I had a couple of apps in the works, which unfortunately I don’t think I’ll have time to finish due to time constraints.
